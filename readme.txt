Dataset link: https://www.kaggle.com/veeralakrishna/cricsheet-a-retrosheet-for-cricket?select=t20s


Follow the following steps to run the project!
1. Download the following modules using pip:
xgboost, numpy, pandas, pickle,sklearn
2. download the dataset from the above link.
3. Create a folder named 'data' and copy contents of folder 'all' to it.
4. run 'data_processing' file
5. run 'final' file
